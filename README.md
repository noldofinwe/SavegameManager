# Save game manager
This is a PBEM game manager for any game that supports hotseat/PBEM.

What it does it syncs a (savegame) directory between you, Azure blob storage and any other players.

## How does it work?
What it does it syncs a (savegame) directory between you, Azure blob storage and any other players.
